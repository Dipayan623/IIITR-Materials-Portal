# IIITR Materials Portal

A centralized academic resource platform built for **IIIT Ranchi** students to access study materials, notes, books, previous year question papers (PYQs), assignments, and lab resources in a structured and user-friendly manner.

## 🌐 Live Demo

https://iiitr-pyq.free.nf/index.php

---

## 📖 Overview

Finding academic resources scattered across different groups and drives can be frustrating. This project aims to solve that problem by providing a single platform where students can easily navigate through their branch, semester, and subjects to access relevant study materials.

The platform also includes a dedicated admin panel that allows administrators to manage academic content without modifying the codebase.

---

## ✨ Features

### Student Portal

* Branch-wise navigation
* Semester-wise organization
* Subject-wise resource management
* Previous Year Question Papers (PYQs)
* Notes and study materials
* Recommended books
* Assignments
* Laboratory resources
* Responsive and easy-to-use interface

### Supported Branches

* Computer Science Engineering
* Computer Science Engineering (DS/AI)
* Electronics and Communication Engineering
* Electronics and Communication Engineering (ES/IoT)
* Mathematics & Computing

### Admin Portal

* Secure administrator login
* Add and manage academic years
* Add and manage branches
* Add and manage subjects
* Upload study materials
* Edit uploaded files
* Delete outdated resources
* Centralized file management dashboard

### Material Categories

* Notes
* Books
* PYQs
* Assignments
* Labs
